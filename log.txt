#Tạo Solution và các Project:
Tạo Blank Solution có tên SV(MSV): ví dụ SV22T1020350
Bổ sung cho Solution các Project sau:
-	<SolutionName>.Admin: Project dạng ASP.NET Core MVC (ví dụ: SV22T1020350.Admin)
-	<SolutionName>.Shop: Project dạng ASP.NET Core MVC
-	<SolutionName>.Models: Project dạng Class Library
-	<SolutionName>.DataLayers: Project dạng Class Library <SolutionName>.BusinessLayers: Project dạng Class Library
